# Задача 07 — Ollama provider

## Цель
Подключить Ollama как альтернативный runtime-провайдер и обеспечить кросс-провайдерную совместимость.

## Инструкция
- Реализовать `OllamaNativeProvider` или `OllamaOpenAiCompatProvider` (первым шагом), затем второй режим.
- Сохранить единый маппинг через core-контракты без дублирования моделей.
- Добавить integration-тесты на общий `ChatRequest` для минимум 2 провайдеров.
- После завершения задачи и мержа PR перенести этот файл из `docs/tasks/todo/` в `docs/tasks/done/`.

## Критерии приемки
- Общий `ChatRequest` работает как минимум на 2 провайдерах.
- Поддержан streaming в реализованном режиме.
- `dotnet test MultiLlm.slnx` проходит.
