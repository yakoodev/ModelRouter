# Задача 02 — Instruction normalizer

## Цель
Обеспечить единый и детерминированный порядок применения инструкций между провайдерами.

## Инструкция
- Реализовать нормализацию слоев `request > session > developer > system`.
- Исключить дублирование логики по провайдерам: нормализация только в core.
- Добавить unit-тесты на порядок и детерминизм.
- После завершения задачи и мержа PR перенести этот файл из `docs/tasks/todo/` в `docs/tasks/done/`.

## Критерии приемки
- Порядок слоев соответствует ТЗ.
- Поведение одинаково для всех провайдеров через единый core-пайплайн.
- `dotnet test MultiLlm.slnx` проходит.
