# Задачи ModelRouter

- `todo/` — активные задачи для выполнения агентом.
- `done/` — завершённые задачи (после выполнения файл задачи переносится из `todo/` в `done/`).

## Правило жизненного цикла задачи
1. Агент берёт задачу из `todo/`.
2. Выполняет реализацию, тесты и обновление документации.
3. После мержа PR переносит файл задачи в `done/` без изменения имени файла.
